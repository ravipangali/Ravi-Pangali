API Documentation

Complete documentation for Firebase notification API endpoints with email/password authentication

Authentication

All API endpoints require email and password authentication. Include your credentials in each request.

Base URL:

https://yourdomain.com/user/api/

This API uses email/password authentication instead of session cookies. Include credentials in every request.

Send Notification

POST

Endpoint

/user/api/firebase/apps/{app_id}/notifications/

Request Body

{
  "email": "user@example.com",
  "password": "userpassword",
  "title": "Notification Title",
  "body": "Notification message content",
  "tokens": ["fcm_token_1", "fcm_token_2"],
  "image_url": "https://example.com/image.jpg",
  "data": {
    "custom_key": "custom_value",
    "action": "open_screen"
  },
  "priority": "high",
  "type": "alarm",
  "sound": "alarm",
  "data_only": false,
  "is_alarm": true,
  "urgent": true,
  "persistent": true
}

Parameters

email* User email address
password* User password
title* Notification title
body* Notification message
tokens Array of FCM tokens (optional)
image_url Image URL (optional)
data Custom data object (optional)
priority "normal", "high", or "urgent" (default: "high")
type Notification type (e.g., "alarm", "alert")
sound Sound to play (e.g., "alarm", "default")
data_only Send only data payload (no notification)
is_alarm Flag for alarm notifications
urgent Flag for urgent notifications
persistent Flag for persistent notifications

Response

Success Response (200)
{
  "success": true,
  "message": "Notification sent successfully",
  "notification_id": "uuid-string",
  "tokens_sent": 5,
  "tokens_delivered": 4,
  "tokens_failed": 1,
  "details": [
    {
      "token": "fcm_token_1",
      "success": true,
      "response": {...}
    }
  ]
}
Error Response (400/401/404)
{
  "success": false,
  "error": "Invalid email or password"
}

Add Device

POST

Endpoint

/user/api/firebase/apps/{app_id}/devices/

Request Body

{
  "email": "user@example.com",
  "password": "userpassword",
  "token": "fcm_device_token_here",
  "device_id": "device_identifier",
  "device_type": "android"
}

Parameters

email* User email address
password* User password
token* FCM device token
device_id Device identifier (optional)
device_type "android", "ios", "web", "unknown" (default: "unknown")

Response

Success Response (200)
{
  "success": true,
  "message": "Device registered successfully",
  "device_id": "uuid-string",
  "created": true,
  "device_type": "android"
}
Error Response (400/401/404)
{
  "success": false,
  "error": "Token is required"
}

Get Devices

GET

Endpoint

/user/api/firebase/apps/{app_id}/devices/list/?email=user@example.com&password=userpassword

Alternative: Headers

mail: user@example.com
X-Password: userpassword

Parameters

email* User email address (query param or header)
password* User password (query param or header)

Response

Success Response (200)
{
  "success": true,
  "devices": [
    {
      "id": "device-uuid",
      "app_name": "My Firebase App",
      "device_type": "android",
      "device_id": "device_identifier",

      "created_at": "2024-01-15T10:30:00Z"

  "success": false,

Usage Examples

JavaScript (Fetch)

// Send notification with email/password auth
const sendNotification = async () => {
  const response = await fetch('/user/api/firebase/apps/your-app-id/notifications/', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      email: 'user@example.com',
      password: 'userpassword',
      title: 'Hello from API!',
      body: 'This is a test notification',
      tokens: ['fcm_token_1', 'fcm_token_2'],
      data: {
        action: 'open_screen',
        screen_id: 'home'
      },
      priority: 'high',
      type: 'alert',
      sound: 'default',
      data_only: false,
      is_alarm: false,
      urgent: false,
      persistent: false
    })
  });
  
  const result = await response.json();
  console.log(result);
};

// Add device
const addDevice = async () => {
  const response = await fetch('/user/api/firebase/apps/your-app-id/devices/', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      email: 'user@example.com',
      password: 'userpassword',
      token: 'fcm_device_token',
      device_id: 'device_123',
      device_type: 'android'
    })
  });
  
  const result = await response.json();
  console.log(result);
};

Error Codes

HTTP Status Codes

400 Request - Missing email/password or required fields
401 Unauthorized - Invalid email/password
404 Not Found - Firebase app not found
200 Success - Request completed successfully

Common Error Messages

"Email and password are required" - Missing authentication credentials
"Invalid email or password" - Authentication failed
"Title and body are required" - Missing required notification fields
"Firebase app not found" - Invalid app ID or app doesn't belong to user
"Account is disabled" - User account is inactive
